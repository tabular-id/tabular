# TABULAR
Simple & Powerfull SQL Client

![Screenshot Halaman Utama](screenshots/halaman-utama.jpg)


## BUILD ON ARCH LINUX
### Requirements
```bash
sudo pacman -Syu clang llvm
sudo pacman -Syu base-devel
```
Export Environment :
```
export LIBCLANG_PATH=/usr/lib
```


## BUILD ON UBUNTU
### Requirements
```bash
sudo apt update
sudo apt install clang libclang-dev
```
